# DoIt Frontend

React 19 + TypeScript + Vite frontend for DoIt `v0.3.0`.

## Commands

```powershell
npm install
npm run dev
npm run build
npm run lint
npm run test
npm run cypress:run
```

## Notes

- Keep the app mobile-first.
- Use Vitest for unit/integration tests.
- Use Cypress for E2E tests.
- Keep visible text ready for i18n.
